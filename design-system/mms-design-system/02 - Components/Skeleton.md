# Skeleton

The `Skeleton` component is used to provide visual feedback during content loading states, reducing perceived wait time.

## 💎 Design Standards
- **Aesthetic**: Pulse animation using alpha-transparency.
- **Radius**: Matches the target component (e.g., `12px` for Cards, `full` for Avatars).

## 🚥 Token Mapping

| Element | Token | Usage |
| :--- | :--- | :--- |
| **Surface** | `var(--surface-component)` | Base skeleton background |
| **Animation** | `pulsate` | 1.5s loop duration |
