# Breadcrumbs

The `Breadcrumbs` component helps users navigate hierarchical structures by showing their current location.

## 💎 Design Standards
- **Aesthetic**: Minimalist inline links.
- **Separator**: `var(--content-subtle)` / Forward Slash or Chevron.
