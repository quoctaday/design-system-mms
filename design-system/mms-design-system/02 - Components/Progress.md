# Progress

The `Progress` component visualizes the status or completion of a task.

## 💎 Design Standards
- **Aesthetic**: Flat track with a brand-colored fill.
- **Micro-interactions**: Smooth width transitions (`transition: width 0.3s cubic-bezier(0.4, 0, 0.2, 1)`).
