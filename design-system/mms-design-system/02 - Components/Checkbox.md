# Checkbox

A control that allows the user to select one or more options from a set.

## 📏 Sizing
Available in two standard operational sizes.

| Prop | Indicator Size | Usage |
| :--- | :--- | :--- |
| `size="1"` | `16px` | Compact lists, data grids |
| `size="2"` | `20px` | Standard forms (Default) |

## ⤗ Radius (Modernized)
Standardized rounding options to match the global UI protocol.

| Prop | Value | Note |
| :--- | :--- | :--- |
| `radius="none"` | `0px` | Sharp corners |
| `radius="sm"` | `8px` | Classic rounded (Default for Checkbox) |
| `radius="md"` | `12px` | Standard rounded |
| `radius="lg"` | `16px` | Premium rounded |
| `radius="full"` | `999px` | Circular checkbox |

## 🎨 Token Specs
- **Color:** Defaults to `brand-primary`. Supports `success`, `error`, `warning`, `gray`.
- **States:** Unchecked, Checked, Indeterminate, Disabled.
- **Focus:** `var(--brand-9)` with 2px offset ring.
