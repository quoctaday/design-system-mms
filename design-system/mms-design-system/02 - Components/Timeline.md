# Timeline

Component used to display events in chronological order.

## 📏 Architecture
Includes the main track, indicator dots, content wrappers, and time stamps.

## 🎨 Token Specs
- **Event Container:** `var(--surface-panel)` (Never hardcoded white, ensuring adaptability across light and dark modes).
- **Track Line:** `var(--border-subtle)`
- **Timestamps:** `var(--content-subtle)`
- **Event Title:** `var(--content-strong)`
