# Sidebar

The `Sidebar` is the primary navigation framework for the MMS Platform.

## 💎 Design Standards
- **Aesthetic**: Flat-Premium panel.
- **Width**: `240px` (Default) or `64px` (Collapsed).
- **Background**: `var(--surface-subtle)` (S2).
