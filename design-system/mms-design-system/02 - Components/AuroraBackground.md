# AuroraBackground

The `AuroraBackground` is a premium, animated background used for high-impact landing areas or hero sections.

## 💎 Design Standards
- **Aesthetic**: Dynamic blurring effect using brand-colored gradients.
- **Micro-interactions**: Subtle motion (`20s+` animation duration).
