# RadioGroup

Standard selection component for picking one option from a set constraint list.

## 🎨 Token Specs
- **Background (Unchecked Inner):** `var(--surface-component)`
- **Background (Checked Inner):** `var(--content-on-solid)` (Ensures distinct checkmark dot stands out natively against dynamic brand colors without resorting to strict white).
- **Border:** `var(--border-default)` transitioning to Brand/Blue when active.
- **Label:** `var(--content-strong)`
