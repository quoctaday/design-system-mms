# Components Introduction

This directory contains the complete atomic and compound component library for the MMS Design System. Every component is engineered for high-density operational workflows and follows the strict **Ironclad: Radix Parity** protocol.

## 🧱 Component Categories

- **Actions:** Buttons, Badges, SegmentedControls.
- **Inputs:** TextFields, Selects, Checkboxes, Switches.
- **Feedback:** Dialogs, Toasts, Tooltips, Progress indicators.
- **Data:** Tables, Timelines, DataLists.
- **Visuals:** Avatars, Auroras, Icons.

## 🛡️ The Ironclad Standard
All components must pass the visual and architectural audit before being codified. This ensures 100% parity with Radix UI and resilient performance across all MMS applications.
