# Toast

The `Toast` component provides brief, non-intrusive feedback about an operation's outcome.

## 💎 Design Standards
- **Aesthetic**: Flat-Premium floating panel.
- **Position**: Bottom-right (Default) or Top-center.
- **Radius**: `12px` (Outer) / `8px` (Inner).

## 🎨 Variants
- **Success**: Checkmark icon + `var(--content-success)`.
- **Error**: Alert icon + `var(--content-danger)`.
- **Info**: Info icon + `var(--brand-9)`.
