# Button

The primary action component for the MMS Platform.

## 📏 Sizing
Buttons follow the standardized height tokens for operational density.

| Prop | Height | Usage |
| :--- | :--- | :--- |
| `size="1"` | `32px` | Micro actions, compact tables |
| `size="2"` | `40px` | Standard UI actions (Default) |
| `size="3"` | `48px` | Primary CTA, Large modals |
| `size="4"` | `56px` | Hero sections, high-impact actions |

## ⤗ Radius (Modernized)
Standardized rounding options to match the global UI protocol.

| Prop | Value | Note |
| :--- | :--- | :--- |
| `radius="none"` | `0px` | Square corners |
| `radius="sm"` | `8px` | Classic rounded (legacy "small") |
| `radius="md"` | `12px` | Standard rounded (legacy "medium") |
| `radius="lg"` | `16px` | Premium rounded (legacy "large") |
| `radius="full"` | `999px` | Pill shape |

## 🎨 Variants
- **Solid:** High-intent actions using Brand Blue (`--surface-solid`).
- **Soft:** Low-emphasis actions with subtle backgrounds.
- **Outline:** Secondary actions with defined borders.
- **Ghost:** Minimal actions within list items or headers.
- **Surface:** Bordered with subtle background for neutral hierarchy.
