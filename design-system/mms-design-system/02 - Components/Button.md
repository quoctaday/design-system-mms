# Button

The primary action component for the Woker Design System.

## 📏 Sizing
Buttons follow the systematic density tokens.

| Prop | Height | Usage |
| :--- | :--- | :--- |
| `size="1"` | `28px` | Micro actions, table rows |
| `size="2"` | `34px` | Standard UI actions |
| `size="3"` | `42px` | Primary CTA, Modals |
| `size="4"` | `50px` | Hero sections |

## 🎨 Variants
- **Primary:** High-intent actions using Brand Blue (`--surface-solid`).
- **Secondary:** Neutral actions using standard component backgrounds.
- **Glass:** Transparent background with backdrop blur for layered interfaces.
