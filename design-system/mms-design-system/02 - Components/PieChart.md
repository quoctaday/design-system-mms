# PieChart

The `PieChart` is a specialized widget for proportional data visualization.

## 💎 Design Standards
- **Aesthetic**: Minimalist flat slices with CSS-based tooltips on hover.
- **Color Scale**: Always use semantic brand tokens or indexed status colors.
