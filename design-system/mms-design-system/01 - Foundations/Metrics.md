# Metrics & Analytics

Rules for data visualization and financial status indicators within the Woker Design System.

## 📊 Chart Tokens
Used for consistent coloring across Pie, Area, and Line charts.

| Role | Target Token | Usage |
| :--- | :--- | :--- |
| **Primary Data** | `--surface-solid` | Main data series |
| **Active Selection** | `--brand-9` | Hovered/Active chart segments |
| **Grid Boundary** | `--border-subtle` | Chart grids and axis lines |
| **Muted Meta** | `--content-subtle` | Axis labels and secondary legends |

## 🟢 Status Indicators
Indicators derive from semantic status colors (e.g., Success-9 for uptrends, Danger-9 for downtrends).

- **Radius**: System default `var(--radius-medium)` (8px).
- **Contrast**: All interactive states must exceed **4.5:1** contrast (WCAG AA).
